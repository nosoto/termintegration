This is a online dashboard reporting incidents for Iberia IT. 
The project was created for my Master in Computer Science at IE University Madrid in March 2022.

The application combines dash and flask. Data is pulled via REST from an Oracle database

To run application
1. Install requirements
2. Run run.py, if needed change port


Application is ready to be deployed on 
1. Azure App Services (via VS Code)
2. Heroku (via command line)
